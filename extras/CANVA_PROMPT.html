Canva Template Creation Prompt for Claude Desktop

Use this prompt with Claude Desktop (which has Canva integration) to create “The AI Architect” slide templates.

Prompt for Claude Desktop:

I need you to create 9 presentation slide templates in Canva for my brand “The AI Architect”. Here are the exact specifications:

Brand Identity

  • Brand Name: The AI Architect
  • Tagline: “Stop Guessing. Start Architecting.”
  • Fonts:
    • Headers: Bebas Neue (or similar condensed sans-serif if unavailable)
    • Body: IBM Plex Sans (or similar clean sans-serif)
  • Color Palette:
    • Navy Deep: #0C2B4E (primary)
    • Navy Royal: #213555 (secondary)
    • Slate Blue: #3E5879 (accent)
    • Cream Gold: #D8C4B6 (highlights)
    • Text Body: #5A6A7A
    • Text Muted: #8896A6
    • Background Light: #FAFBFC
    • Background Warm: #F8F6F3

Slide Dimensions

  • Size: 1600 × 2000 px (portrait, 4:5 ratio)
  • This allows for Instagram posts, LinkedIn carousels, and presentation slides

OPTION A: Minimal Blueprint Style

Design Elements for All Option A Slides:

  • Background: #FAFBFC with subtle grid pattern (20px × 20px squares at 3% opacity in Navy Royal)
  • Top accent bar: 4px tall, full width, Navy Deep (#0C2B4E)
  • Header section: “THE AI ARCHITECT” in 11px uppercase, 3px letter spacing, Navy Deep
  • Page numbers: 10px, top right, Text Muted (#8896A6)
  • Sharp corners (border-radius: 0) on all elements

A1. Header Slide

Layout: - Top bar (4px, Navy Deep) - Header: “THE AI ARCHITECT” (top left, 32px from top) - Page number: “01” (top right, 32px from top) - Center content (vertically centered): - Eyebrow text: “FRAMEWORK 001” (10px, uppercase, 3px letter spacing, Text Muted) - Main title: “AI-ART MATRIX” (80px, Bebas Neue, Navy Deep, -1px letter spacing) - Divider line: 60px × 2px, Cream Gold, centered below title - Subtitle: “The operating system for AI-enabled enterprises.” (13px, Text Body, max-width 280px, centered)

A2. Content Slide

Layout: - Top bar + header (same as A1) - Section title: “THE PROBLEM” (48px, Bebas Neue, Navy Deep) - Accent line: 40px × 2px, Cream Gold, below title - Body text: 2-3 paragraphs, 14px, line-height 1.7, Text Body - Highlight box (at bottom): - Background: Cream Gold at 10% opacity - Border-left: 3px solid Cream Gold - Padding: 16px - Strong text: “KEY INSIGHT” (13px, Navy Deep, bold) - Description text: 13px, line-height 1.6, Text Body

A3. Bullets Slide

Layout: - Top bar + header (same as A1) - Section title: “FOUR CORE PILLARS” (40px, Bebas Neue, Navy Deep) - Accent line: 40px × 2px, Cream Gold - 4 bullet points with: - Square bullet icon: 8px × 8px, Navy Deep - Text: 14px, line-height 1.6, Text Body - Bold label followed by description - 16px gap between bullets

OPTION B: Dark Authority Style

Design Elements for All Option B Slides:

  • Background: Navy Deep (#0C2B4E) with grid pattern (24px × 24px at 2% white opacity)
  • Corner accents: 80px × 80px L-shaped borders (1px, Cream Gold at 30% opacity)
    • Top-right corner (missing left and bottom borders)
    • Bottom-left corner (missing right and top borders)
  • Header: “THE AI ARCHITECT” (10px, 4px letter spacing, Cream Gold)
  • Square marker: 8px × 8px, Cream Gold (top right)

B1. Header Slide

Layout: - Corner accents (top-right, bottom-left) - Header section + marker - Center content: - Eyebrow: “STRATEGY FRAMEWORK” (10px, uppercase, 3px letter spacing, white at 40% opacity) - Title: “THE AI-ART MATRIX” (72px, Bebas Neue, white, with “AI-ART” in Cream Gold) - Horizontal line: 80px × 1px, Cream Gold, centered - Subtitle: “Operational leverage for Fortune 500 enterprises.” (13px, white at 60% opacity, centered, max-width 280px)

B2. Content Slide

Layout: - No corner accents - Header with border-bottom: 1px solid white at 10% opacity - Page indicator: “02 / 05” (top right, 10px, white at 40% opacity, 2px letter spacing) - Section title: “WHY IT MATTERS” (48px, Bebas Neue, white) - Gold gradient accent: 60px wide × 2px, gradient from Cream Gold to transparent - Body text: 2-3 paragraphs, 14px, line-height 1.8, white at 70% opacity - Callout box: - Background: Cream Gold at 10% opacity - Border: 1px solid Cream Gold at 30% opacity - Padding: 20px - Strong label: “CASE STUDY” (13px, uppercase, 1px spacing, Cream Gold, bold) - Text: 13px, white at 80% opacity, line-height 1.6

B3. Bullets Slide

Design Elements for All Option C Slides:

  • Background: Warm (#F8F6F3)
  • Left sidebar: 48px wide, Navy Deep, full height

  • Main content area: starts 48px from left edge

C1. Header Slide

Instructions for Creating in Canva:

  1. Create 9 blank designs (1600 × 2000 px each)
  2. For each design:
    • Set the background color
    • Add grid pattern using Canva’s pattern/texture overlays (or create with lines)
    • Add all text elements with exact fonts, sizes, and colors
    • Add decorative elements (bars, boxes, corners, etc.)
    • Ensure all corners are sharp (no rounding)
    • Use exact spacing and padding specified
  3. Save each template with descriptive names:
    • “AI-Architect-A1-Header”
    • “AI-Architect-A2-Content”
    • “AI-Architect-A3-Bullets”
    • (Same pattern for B and C options)
  4. Make them editable templates so I can easily swap text/content

Sample Content for Testing:

Header Slides: “AI-ART MATRIX”, “THE OLCD PROTOCOL”, “OPERATIONAL LEVERAGE” Content Slides: 2-3 paragraphs about AI strategy, implementation, or case studies Bullets Slides: Framework steps, key principles, or implementation checklist

Please create all 9 templates following these exact specifications. Maintain consistency within each design option (A, B, C) while ensuring they’re distinct from each other.
